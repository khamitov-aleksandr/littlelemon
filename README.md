# React Page for the "Little Lemon restaurant"

## About the Project
This is a responsive React webpage for the "Little Lemon restaurant". You can see menu, write a feedback, login, book a table and read more about restaurant. Also here is wireframes from Figma to see how was UI/UX design planned.


## Technologies
* React.js, HTML, CSS, JS
* Figma
* VS Code
* Git & GitHub
